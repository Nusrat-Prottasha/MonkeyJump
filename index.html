Monkey Jump: MoE-Style PEFT for Efficient Multi-Task Learning

MoE-style specialization for PEFT without trainable routers. Gradient-free routing via k-means + EMA-updated centers.
Website Code arXiv

Abstract

Mixture-of-experts (MoE) variants of PEFT enable per-token specialization but introduce extra trainable routers and expert parameters. We propose Monkey Jump (MJ), which achieves MoE-style specialization while adding zero extra trainable parameters. MJ reuses the adapters already present in each Transformer block as implicit experts and performs gradient-free routing using $k$-means with EMA-updated centers.

How to use Monkey Jump

from transformers import AutoModelForCausalLM
from src.MJLoRA import apply_monkeyjump

model = AutoModelForCausalLM.from_pretrained("model_name")

model = apply_monkeyjump(
    model,
    blocks={"LlamaDecoderLayer": list(range(32))},
    linears=["q_proj", "k_proj", "v_proj", "o_proj"],
    shared_expert=["up_proj", "down_proj"],
    rank=2,
    alpha=16.0,
    temperature=1.0,
    ema_momentum=0.2,
    top_k=2,
    rep_mode="token",
)
